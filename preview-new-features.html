Preview delle 4 nuove feature

Questa pagina mostra come appariranno le nuove feature integrate nell'app:

  • 1. Earnings Calendar Widget (banner viola nella header card)
  • 2. Indicatori Tecnici Card (con RSI gauge, Moving Averages, Support/Resistance)
  • 3. Dividendi Timeline (storico + metriche + prossima data)
  • 4. Heatmap Settoriale (treemap interattiva con performance per settore)

Tutte le feature mantengono lo stile esistente dell'app: stesso schema colori (slate/blue), stesse card arrotondate, stesso font Inter, stessi componenti UI.
